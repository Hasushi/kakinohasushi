# 🍣 Portfolio Project: kakinohasushi

「Hasu」による、回転寿司をテーマにした遊び心あふれるポートフォリオサイト。
柿の葉寿司の伝統（職人技）と、回転寿司の現代的な楽しさを融合させたデジタル体験。

---

## 🎨 コンセプト
- **店名**: kakinohasushi (柿の葉寿司)
- **テーマ**: おもてなし（UI/UX）と職人技（技術力）の融合
- **モチーフ**:
  - **柿の葉**: 制作物（実績）を包む、開く楽しみの象徴
  - **回転レーン**: 絶え間なく流れるインスピレーションとキャリアの歩み
  - **タッチパネル**: インタラクティブな自己紹介・経歴紹介

---

## 🗺️ サイト構成案

### 1. 暖簾（Entrance）
- イントロダクション・アニメーション。
- 「暖簾」をくぐって（クリックして）入店する演出。

### 2. 回転レーン（Main Contents）
- **実績（ネタ）**: 無限に流れるレーンに乗った制作物。
- **ギミック**: 皿をホバーすると柿の葉が開き、作品のプレビューが表示される。
- **詳細**: クリックするとお品書き（詳細ページ）へ遷移。

### 3. 注文用タブレット（Profile / Career）
- **店主紹介**: Hasuのプロフィール、板前としての心得。
- **修行の記録**: 経歴を年表形式で（皿のランクで経験値を表現）。
- **秘伝のタレ**: 技術スタック（スキルセット）の紹介。

### 4. ビッくらポン！（Gamification）
- 特定の皿（実績）を一定数見ると発生するミニゲーム。
- 当たりが出ると「隠しスキル」や「裏メニュー（秘密の趣味）」が解禁。

---

## 🛠️ 技術スタック (Technical Stack)

| カテゴリ | 選定技術 | 理由 |
| :--- | :--- | :--- |
| **Framework** | **Next.js (App Router)** | 高速な動作、SEO対応、スケーラビリティ。 |
| **Styling** | **Tailwind CSS** | 和風カラーや木目調のカスタムデザインの効率化。 |
| **Animation** | **Framer Motion** | 回転レーンの無限ループ、皿のバウンド、葉の開閉演出。 |
| **State Management** | **Zustand** | 食べた皿のカウント、ゲームのフラグ管理など。 |
| **3D Elements** | **React Three Fiber** | 柿の葉寿司の3Dモデル操作（オプション）。 |
| **Deployment** | **Vercel** | Next.jsとの親和性とデプロイの容易さ。 |
